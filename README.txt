========================================
  WUBSTATION 9000 v6
  Browser Bass Synthesizer by Bill Barth
========================================

WHAT IS THIS?
  A full-featured bass synthesizer that runs entirely in your web browser.
  No installation required. No accounts. No internet needed after first load.

HOW TO RUN IT:
  1. Unzip this folder somewhere on your computer
  2. Open "index.html" in a modern web browser (Chrome or Edge recommended)
  3. Click "GENERATE WUB" at the bottom to start making sounds
  4. Make some wubs!

TIPS:
  - Use Chrome or Edge for best performance (Firefox works but may have latency)
  - Use headphones or good speakers to hear the full bass range
  - Click the keys at the bottom to play notes, or use your computer keyboard
  - Try the presets to hear what it can do
  - The sequencer lets you build patterns

FEATURES:
  Sub-oscillator, FM synthesis, noise generator, unison, filter envelope,
  bitcrusher, phaser, chorus, delay, reverb, vowel filter, ring modulator,
  comb filter, stutter, glide, ADSR, oscilloscope, MIDI support,
  step sequencer, and WAV export.

REQUIREMENTS:
  - A modern web browser (Chrome, Edge, Firefox, or Safari)
  - That's it!

TROUBLESHOOTING:
  - No sound? Make sure you clicked "GENERATE WUB" first
  - Still no sound? Check your system volume and make sure the right output device is selected
  - Glitchy audio? Close other tabs/apps to free up CPU
